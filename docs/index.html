실측 인쇄

입력한 명패 가로/세로 mm 값을 WPF 출력 단위로 변환해 실제 크기에 맞춰 인쇄합니다.

여러 명패 배치

A4 등 선택한 용지와 여백, 간격을 기준으로 한 장에 들어가는 개수를 자동 계산합니다.

항목별 디자인

참석자마다 표시명, 소개 문구, 소속/메모, 배경 이미지, 로고 이미지를 다르게 설정할 수 있습니다.
