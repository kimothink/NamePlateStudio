NamePlateStudio

아크릴 명패 안쪽에 넣을 이름표를 직접 꾸미고, 입력한 실제 mm 크기로 A4 용지에 여러 개 배치해 출력하는 Windows 프로그램입니다.

실측 인쇄

입력한 가로, 세로 mm 값을 WPF 인쇄 단위로 변환해 실제 크기에 맞춰 출력합니다.

A4 여러 개 배치

용지 크기, 여백, 간격을 기준으로 한 장에 들어갈 명패 개수를 자동 계산하고 미리 볼 수 있습니다.

앞면과 뒷면 편집

항목별 이름, 소속 문구, 뒷면 문구, 배경 이미지, 색상, 회전과 크기를 다르게 설정할 수 있습니다.

Windows 설치 파일

NSIS로 만든 win-x64 self-contained 설치 파일입니다. 별도 .NET 런타임 설치 없이 실행할 수 있습니다.
